Finger Strength Analyzer

An interactive, web-based tool to analyze tindeq finger board data.

 What you can do with this application:

  • Visualize tindeq outputs. 
  • Ready to send? Compare your realtime output to your past performance history. 
  • Making progress? Track changes in your performance overtime. 

 How this application works:

  • Users upload data measured by tindeq to the application. 
  • The application crunches some numbers and returns the user with usefull perfromance statistics. 
  • Tindeq data and statistics are returnted to the user via interactive visualizations. 

I hope this application helps you crush harder!! 🧗 ⛰ ⛰ 🤙 🤙

 How does this application use my (your) data?

  • This application does not save your data in anyway. 
  • Once you close the application your data dies with the session.
